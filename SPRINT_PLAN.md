# Game Programming Weekly Sprint Plan

Course: Game Programming (GP)  
Instructor: Dr. Amna Lodhi  
Sprint Period: June 15-19, 2026

This repository is organized according to the weekly sprint assignment. Each sprint folder contains the related project files and a short README explaining the work completed for that sprint.

## Sprint Overview

| Date | Sprint | Focus | Deliverable |
| --- | --- | --- | --- |
| June 15, 2026 | Sprint 1 | Modeling | Tile prefab and core gameplay object structure |
| June 16, 2026 | Sprint 2 | Texturing | Snake, fruit, board, icon, and UI texture assets |
| June 17, 2026 | Sprint 3 | Importing | Imported Unity assets organized by type |
| June 18, 2026 | Sprint 4 | Level Design | Main Unity scene and playable board prefab setup |

## Notes

The original assignment describes 3D modeling, UV mapping, and rendering. This project is a 2D Unity Snake game, so the sprint deliverables have been adapted to the project type:

- 2D prefabs and board objects are used for modeling.
- Sprite images are used for texturing.
- Unity asset folders represent the import pipeline.
- The Main scene represents level design.
- C# scripts represent gameplay implementation.
- Screenshot and quality settings represent final rendering output.
